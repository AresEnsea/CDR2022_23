1ere chose à faire est :	git pull

Si vous ajoutez un changement : 	git add * 					(* signifie que vous ajouté tout ce que vous avez modifié)
		 		git commit -m "raison de la modif"
		 		git push

Si vous supprimez : 		git rm monFichier.txt
				git commit -m "raison de la suppresion"
				git push

Si vous êtes à jour :		git status

Guides des commandes :		https://www.hostinger.fr/tutoriels/commandes-git#Git_config

Si vous voulez travailler à plusieurs sur un même fichier utilisez les "branch"
